# Pastelería Mil Sabores

Este repositorio contiene el proyecto web de la pastelería "Mil Sabores". El objetivo es construir una página web estática con la estructura de secciones necesarias para mostrar información como productos, contacto, nosotros y gestión de usuario.

## Estructura del proyecto

- `index.html` — Página principal del proyecto.
- `src/components/` — Vistas parciales o páginas secundarias.
- `src/css/` — Estilos CSS organizados por sección.
- `src/js/` — Scripts JavaScript por sección.
- `src/assets/` — Recursos multimedia como imágenes y logos.

## Tecnologías

- HTML5
- CSS
- JavaScript

## Cómo usar

1. Abrir `index.html` en un navegador.
2. Navegar entre las páginas y secciones disponibles.
